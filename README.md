# rabbitmq-demos

 RabbitMQ常用方法用例。

## 开发环境

- JDK 1.8
- RabbitMQ 3.9 （Docker-Desktop-windows/Docker/Linux）
- Maven 3.6.3

## 项目详细

| 模块名               | 内容                                        |
| -------------------- | ------------------------------------------- |
| rabbitmq-basic       | RabbimtMQ Hello world,Work Queue,Routing... |
| rabbitmq-dlx         | RabbitMQ 死信队列                           |
| rabbitmq-delay-queue | RabbitMQ 延迟队列                           |
